Earthslime

Earthslime is a portable piston, ideal for simple flying carpets. Well, assuming bricks count as carpet.

Entity Effects:

  • +5 seconds Slowness

  • Pulls target

Block Effects:

  • Pulls block

Skyslime

Entity Effects:

  • +5 seconds Slowness

  • Pushes target

Block Effects:

  • Pushes block
