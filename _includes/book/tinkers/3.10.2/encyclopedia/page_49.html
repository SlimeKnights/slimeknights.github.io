Sinistral

Causes the crossbow to fire on attack instead of interact, and to only load when not sneaking. Allows using an item in the offhand while using a crossbow.

Effects:

  • Single level
  • Requires 1 upgrade slot

Trueshot

Grants +0.1 accuracy per level.

Effects:

  • Can apply levels incrementally
  • Maximum of 3 levels
  • Requires 1 upgrade slot per level
