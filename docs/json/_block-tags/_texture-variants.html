* `tconstruct:planklike`: Wood blocks in this tag are valid options for the part builder and tinker station. Typically includes wood planks. * `tconstruct:workstation_rock`: Stone blocks in this tag are valid options for crafting the modifier workstation. Typically includes smooth forms of stone. * `tconstruct:anvil_metal`: Resource blocks in this tag are valid options for crafting the Tinkers' Anvil. Typically includes metal alloys.
