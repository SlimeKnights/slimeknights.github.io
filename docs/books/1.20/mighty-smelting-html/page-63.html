Multishot

Thruul is not responsible for any accidents while shooting more than one projectile.

Effects:

  • Shoots multiple arrows at once
  • Multiple levels, each level increases amount of arrows fired
  • Requires 1 ability slot per level

Returning

Like Mjolnir!

Effects:

  • Causes thrown tool to return after a short time
  • Higher levels return faster
  • Requires 1 ability slot for first level
  • Higher levels require 1 upgrade slot
  • Requires throwing or a ballista bow
