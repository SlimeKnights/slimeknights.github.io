Pathing

If you can't find the golden brick road make your own dirt path road.

Effects:

  • Allows you to make paths with your tools
  • Single level
  • Requires 1 ability slot

Slurping

Mmm, delicious!

Effects:

  • Drink fluids stored in the tool on usage
  • Requires 1 ability slot per level
