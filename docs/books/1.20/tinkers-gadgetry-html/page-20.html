Slimelytra

Tired of walking? Try flying with these futuristic wings!

To create these futuristic wings, you simply need to pour enderslime on an elytra. The slimelytra can later be repaired using phantom membranes.

Properties:

  • High Durability
  • No Armor or Defense Slots
  • 5 Upgrade Slots
  • 1 Ability Slot

Slimeshell

Slimeshells are one part defense, one part storage, granting the best of both shulker abilities.

To create these wondrous pants, pour enderslime on a shulker shell in a basin. The shell can later be repaired using chorus.

Properties:

  • High Durability
  • No Armor or Defense Slots
  • 5 Upgrade Slots
  • 1 Ability Slot
  • Shulking
