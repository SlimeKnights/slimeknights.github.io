Item Frame

Adds 3 slots per level to the helmet which will render on the UI when the helmet is worn.

Helmet inventory can be opened by interacting with it while holding.

Effects:

  • Requires 1 upgrade slot per level

Minimap

Adds 3 slots per level for maps to view while wandering. Use control plus helmet interact to swap the active map.

Helmet inventory can be opened by interacting with it while holding.

Effects:

  • Requires 1 upgrade slot per level
