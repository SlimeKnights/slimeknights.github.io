Rebalanced

Grants +1 modifier slot; type is determined by the recipe. Applying +1 ability slot makes the tool lose 1 upgrade slot.

Enderslime increases the level of one of the tool traits, typically the head trait.

Effects:

  • Single use
  • Slotless

Recapitated

Grants +1 upgrade slot.

Effects:

  • Single use
  • Slotless
