Barebow

Prevents the bow from using ammo from the inventory. Useful to ensure that crystalshot or bulk quiver is used instead, or to otherwise use a bow for firing things besides arrows.

Effects:

  • Single use
  • Slotless

Blindshot

Decreases accuracy by 0.1 per level.

Effects:

  • Multi-use
  • Slotless
  • No max level, though accuracy minimum is 0.1
