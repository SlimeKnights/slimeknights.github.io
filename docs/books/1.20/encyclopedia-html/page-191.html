Bounce

Causes arrows to bounce after hitting a block instead of stopping. First level allows a single bounce, each further level adds up to 2 more bounces.

Effects:

  • Maximum of 3 levels
  • Requires 1 upgrade slot per level

Collecting

Allows reeling in more objects with the fishing rod. Items, XP orbs, arrows, and thrown tools can all be hooked and reeled in.

Successfully reeled in objects will be moved directly to the player.

Effects:

  • Single level
  • Requires 1 upgrade slot
