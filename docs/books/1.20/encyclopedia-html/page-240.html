Blocking

Allows using the tool as a shield, blocking many potential attacks. Can often block while charging the tool. Most tools can block up to 5 damage, but some (such as staffs) can block amounts in the range of 15 to 20 damage.

Effects:

  • Single level
  • Requires 1 ability slot

Dual Wielding

Allows the tool to be used from both the main and the offhand. Tools with dual wielding have 80% attack speed and damage, or 90% attack speed and 70% damage for two handed tool.

Two handed tools must first receive offhanded in order to receive dual wielding.

Effects:

  • Maximum of 1 level
  • Requires 1 ability slot
