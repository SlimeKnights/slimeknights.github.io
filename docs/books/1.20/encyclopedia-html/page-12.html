Fishing Rod

The Fishing Rod is a short-distance ranged weapon that requires no ammo. It damages reeled in monsters, and is useful for fishing.

When using interaction modifiers, by default the fishing rod acts on left click, though a modifier worktable can change that.

Properties:

  • 2.0 Attack Speed
  • 1.0 Draw Speed
  • 1.00 Arrow Velocity
  • 0.75 Accuracy
  • 150% Durability Multiplier
  • Catches fish

Javelin

The Javelin is a hybrid melee/ranged weapon that can be used for both close ranged attacks and thrown to attack at a distance.

Properties:

  • +2 Attack Damage
  • 1.5 Attack Speed
  • Sums stats of both all parts
  • Throwing
