Enderslime

Teleporting your problems to a random location is a method of dealing with them.

Entity Effects:

    +7 Seconds Returning

    Teleports target if no current Returning

Entity Effects:

    Replaces hit block with held block

Molten Slimesteel

Slimesteel is ideal for anyone wishing to fly; as long as you found a way to deal with the landing.

Entity Effects:

    Power 1.5 explosion

    0 Explosion Damage

    Extra explosion knockback

Entity Effects:

    Explosion breaks blocks
