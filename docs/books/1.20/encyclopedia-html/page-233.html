Drill Attack

Hitting a mob within the first second of a movement ability such as springing, flinging, or grapple will cause them to be attacked with the main hand weapon.

Effects:

  • Maximum of 1 level
  • Requires 1 ability slot
  • Requires springing, flinging, or grapple

Firestarter

Allows the tool to create fires on use, consuming 1 durability.

Effects:

  • Maximum of 1 level
  • Requires 1 ability slot
