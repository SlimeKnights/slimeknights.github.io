Workbench

Adds a 2x2 crafting table to the tool's inventory.

Effects:

  • Single level
  • Slotless

Worldbound

Alternative to netherite that makes tools invulnerable when dropped without requiring an upgrade slot. Does not grant any stat boosts

Effects:

  • Single use
  • Slotless
