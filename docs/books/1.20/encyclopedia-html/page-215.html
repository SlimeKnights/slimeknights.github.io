Offhanded

Prevents using the main hand's interaction ability if the offhand is occupied. For example, an axe will not strip wood if a shield is in the offhand.

The second level entirely disables the tool interaction ability.

Effects:

  • Maximum of 2 levels

Overslime

Overslime grants tools a second durability bar, used before the main durability bar. By default, tools start with 50 max overslime, multiplied by its durability modifier.

Tools with no overforced or slimy materials have reduced stats. Slime bottles can be used in the crafting table to restore overslime.

Effects:

  • Multi-use
  • Slotless
  • Non-slimy tools: -10% mining speed and attack damage, -0.5 armor
