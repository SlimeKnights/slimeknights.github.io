Pathing

Allows the tool to create paths, similarly to pickadzes or excavators. Tools will use their normal mining area for placing paths.

Triggers before the tool's native stripping or tilling if present.

Effects:

  • Maximum of 1 level
  • Requires 1 ability slot

Slurping

Allows drinking fluid contained in the tool to perform effects, such as lighting yourself on fire, clearing status effects, or restoring hunger.

Higher levels consume more fluid to perform stronger effects

Effects:

  • No max level
  • Requires 1 ability slot per level
  • Fill by using tool on a fluid tank
