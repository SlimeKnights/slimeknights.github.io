Haste

On harvest tools grants +4 mining speed per level.

Effects:

  • Can apply levels incrementally
  • Maximum of 5 levels
  • Requires 1 upgrade slot per level

Hydraulic

Grants +4 mining speed per level in the rain. When underwater, instead grants +8 mining speed per level, bypassing the normal debuff from mining underwater.

Effects:

  • Can apply levels incrementally
  • Maximum of 5 levels
  • Requires 1 upgrade slot per level
