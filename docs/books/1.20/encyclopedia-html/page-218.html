Worldbound

Alternative to netherite that makes tools invulnerable when dropped without requiring an upgrade slot. Does not grant any stat boosts

Effects:

  • Single use
  • Slotless

Bonus

Bonus modifiers are a group of slotless modifiers that grant additional modifier slots.

By default, most small tools start with 3 upgrade slots, and most broad tools start with 2. Modifiers can add an additional 4 slots to the tool.

In addition, most tools start with 1 ability slot. Modifiers can add an additional ability slot to the tool, or potentially even an additional two.
