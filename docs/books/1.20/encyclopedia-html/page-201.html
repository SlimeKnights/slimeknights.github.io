Knockback

Grants 1 knockback per level to all weapons, which launches a mob roughly 2 blocks. Most tools start with 0.4 knockback.

Effects:

  • Maximum of 3 levels
  • Requires 1 upgrade slot per level

Sleeves

Gives the chestplate 3 inventory slots per level to hold throwable items. Items in sleeves can be thrown by interacting with an empty hand. Can sneak while interacting to change the selected item.Inventory can be accessed by right clicking the chestplate from an inventory.

Effects:

  • Maximum of 3 levels
  • Requires 1 upgrade slot per level
