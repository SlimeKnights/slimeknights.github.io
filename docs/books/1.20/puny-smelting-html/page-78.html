Speedy

Gotta go fast.

Effects:

  • Increases movement speed
  • Multiple levels, each level increases speed further
  • Requires 1 upgrade slot per level

Step Up

Put one foot in front of the other, except higher.

Effects:

  • Allows climbing taller blocks like stairs
  • Multiple levels, increases step height with each level
  • Requires 1 upgrade slot per level
