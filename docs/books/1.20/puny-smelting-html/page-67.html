Armor Pierce

Tool gains the power of the cactus, attacking through armor.

Not to be confused with impaling

Effects:

  • Arrows cancel out a small amount of armor
  • Multiple levels, each level increases effect
  • Requires 1 upgrade slot per level

Arrow Pierce

I'd be rather appalled if you impaled me with impaling.

Not to be confused with antiaquatic

Effects:

  • Allows arrows to keep going after hitting a target
  • Multiple levels, each can pierce another target
  • Requires 1 upgrade slot per level
