Smelting

How hard must you slap a chicken to cook it?

Effects:

  • Sneak while interacting to fill tool with items
  • Using the tool cooks the items
  • Multiple levels, each level adds another slot
  • Requires 1 upgrade slot per level

Soulbound

Infusing an object with your soul doesn't sound healthy. But it is rather useful!

Effects:

  • Tool will remain with you upon an untimely demise
  • Single Level
  • Requires 1 upgrade slot
